Product Catalog Manager

A modern Flutter application for managing and displaying product catalogs efficiently.  
This project is designed with clean UI, responsive layouts, and smooth user experience.

---

## 🚀 Features

- 📦 Product Listing
- 🔍 Search Products
- 🗂️ Category Management
- ❤️ Favorite Products
- 📱 Responsive Flutter UI
- ⚡ Fast Performance
- 🎨 Clean and Modern Design

---
